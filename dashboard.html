    🎨 Réglage du Thème (Crucial)

    Attention : Ne pas utiliser de thème

    Pour que l'effet Liquid Glass et les transparences fonctionnent, vous ne devez appliquer aucun thème sur votre page ou votre profil.

    Réglage requis : Sélectionnez "Home Assistant" ou "Utiliser le thème par défaut" dans vos paramètres. Pourquoi ? Les thèmes tiers bloquent les modifications CSS de card-mod et cassent le design visuel des cartes.

    🛠️ Prérequis Généraux

    Pour utiliser ces cartes, assurez-vous d'avoir installé les éléments suivants via HACS :

    • button-card Le moteur principal pour le design et les animations.
    • card-mod Indispensable pour les effets (CSS).
    • Mushroom Cards Pour des icônes et un style moderne épuré.
    • vertical stack In Card Pour grouper vos cartes sans bordures visibles.
    • Mini graph card Pour des graphiques compacts et stylés.

    Astuce

    Après l'installation, faites CTRL + F5 pour actualiser le cache.

    TIP

    Toutes les cartes sont compatible en mode sombre comme en mode clair (Sauf indication contraire).
